# POS-System
**In Progress**
Simple Point of Sale system in terminal for learning purpose.
All code is constrained from what learned in classes.

## Installation
If in linux, just clone and compile it. Make sure **JDK** is installed.
```sh
git clone https://github.com/amirulsh/POS-System
cd POS-System
javac -d out src/**/*.java
```
Run the compiled code.
```sh
java -cp out POS
```
